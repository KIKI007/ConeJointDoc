Welcome to Lumache's documentation!
===================================

**ConeJoint** (/lu'make/) is a C++ library for designing structurally stable assmeblies using Cone-like joints, developed from the paper "MOCCA: Modeling and Optimizing Cone-joints for Complex Assemblies".

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
